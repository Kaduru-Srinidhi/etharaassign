# Team Task Manager - Project Submission

## 📋 Project Overview

**Team Task Manager** is a full-stack collaborative task management web application with role-based access control, built with modern web technologies.

### ✅ What's Included

#### 📁 Source Code
- **Backend** (Node.js + Express + PostgreSQL)
  - REST APIs for authentication, projects, and tasks
  - JWT token-based security
  - Database initialization and migrations
  - Error handling and validation
  - Comprehensive routing and middleware

- **Frontend** (React 18 + Vite)
  - Modern responsive UI with CSS3
  - React Router for navigation
  - Axios for API communication
  - Component-based architecture
  - Dashboard and Kanban board views

## 🚀 Getting Started

### Option 1: Local Development (5 minutes)

```bash
# 1. Navigate to project
cd etharaassignment

# 2. Copy backend environment
cd backend
cp .env.example .env
# Edit .env with your PostgreSQL credentials

# 3. Copy frontend environment  
cd ../frontend
cp .env.example .env
# Uses default localhost settings

# 4. Install dependencies (from root)
./setup.sh  # Unix/macOS
setup.bat   # Windows

# 5. Start backend (Terminal 1)
cd backend && npm run dev

# 6. Start frontend (Terminal 2)
cd frontend && npm run dev

# 7. Visit http://localhost:3000
```

See **SETUP_AND_DEPLOYMENT.txt** for detailed local setup.

### Option 2: Deploy to Railway (Production)

1. Create Railway account at https://railway.app
2. Follow detailed steps in **RAILWAY_DEPLOYMENT.txt**
3. Set up PostgreSQL service
4. Deploy backend service with environment variables
5. Deploy frontend service with API URL
6. Test live application

Estimated deployment time: **15-20 minutes**

## 📡 API Endpoints

### Authentication (5 endpoints)
- `POST /api/auth/signup` - Register
- `POST /api/auth/login` - Login
- `GET /api/auth/me` - Get current user

### Projects (7 endpoints)
- `POST /api/projects` - Create
- `GET /api/projects` - List
- `GET /api/projects/:id` - Get details
- `PUT /api/projects/:id` - Update
- `DELETE /api/projects/:id` - Delete
- `POST /api/projects/:id/members` - Add member
- `GET /api/projects/:id/members` - Get members

### Tasks (6 endpoints)
- `POST /api/tasks/:projectId/tasks` - Create
- `GET /api/tasks/:projectId/tasks` - List
- `PUT /api/tasks/:projectId/tasks/:taskId` - Update
- `DELETE /api/tasks/:projectId/tasks/:taskId` - Delete
- `GET /api/tasks/dashboard/overview` - Dashboard

**Total: 18 REST API endpoints**
