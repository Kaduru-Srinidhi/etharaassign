# Team Task Manager - Testing & Demo Guide

## 🎬 Demo Walkthrough Script

### Scene 1: User Authentication (30 seconds)
**What to show**: Sign up and login functionality

1. **Signup Page**
   - Click "Sign up" link
   - Fill in form:
     - Name: "John Doe"
     - Email: "john@example.com"
     - Password: "password123"
   - Click "Sign Up"
   - Show successful login redirect to dashboard

2. **Login**
   - Go to login page (or logout first)
   - Show login with same credentials
   - Explain JWT token storage in localStorage

**Key Points**: 
- Secure password hashing with bcryptjs
- JWT token-based authentication
- Session persists across page refresh

---

### Scene 2: Project Management (1 minute)
**What to show**: Creating and managing projects

1. **Create Project**
   - On Dashboard, show project creation form
   - Enter:
     - Name: "Website Redesign"
     - Description: "Complete redesign of company website"
   - Click "Create Project"
   - Show new project in projects list

2. **View Projects**
   - Click on project card
   - Show project details page with all sections
   - Explain project creation by admin

**Key Points**:
- Project isolation (only members can access)
- Project-level permissions
- Admin automatically added to new projects

---

### Scene 3: Task Management (1.5 minutes)
**What to show**: Creating and managing tasks with kanban board

1. **Create Task**
   - In project, show "Create Task" form
   - Fill in:
     - Title: "Update homepage design"
     - Description: "Make design responsive"
     - Priority: "High"
     - Due Date: (select future date)
     - Assign to: (select team member)
   - Click "Create Task"
   - Show task appears in "To Do" column

2. **Update Task Status**
   - Drag task from "To Do" → "In Progress" (or click arrow)
   - Show task moves to middle column
   - Drag from "In Progress" → "Done"
   - Show task fades in "Done" column

3. **Create More Tasks**
   - Create 2-3 more tasks with different priorities
   - Show color-coded priority badges
   - Show team member assignment indicators

**Key Points**:
- Kanban board for visual task management
- Status workflow: To Do → In Progress → Done
- Priority levels affect task importance
- Team member assignment and tracking
- Task deletion capability

---

### Scene 4: Team Management (45 seconds)
**What to show**: Adding team members with roles

1. **Show Team Members Section**
   - In project, show project members list
   - Explain current members and their roles
   - Show project creator has "admin" role

2. **Add New Member**
   - (Note: Would need another user account for demo)
   - Show form to add member:
     - User ID: (enter user id)
     - Role: Select "Member" or "Admin"
   - Click "Add Member"
   - Show member appears in list

**Key Points**:
- Role-based access control
- Admin can manage team members
- Members limited to assigned tasks
- Project isolation ensures security

---

### Scene 5: Dashboard Overview (45 seconds)
**What to show**: Dashboard statistics and insights

1. **Dashboard Stats**
   - Show statistics cards:
     - Total Tasks

     - Overdue Tasks (if any)

   - Explain data is personalized

2. **Task Lists**
   - Show assigned tasks with projects
   - Show overdue tasks highlighted in red
   - Explain sorting by due date

**Key Points**:
- Overview of key metrics
- Quick identification of overdue work
- Organized task management
- Project quick access

## 💬 Common Questions & Answers

**Q: Can I use this for personal tasks?**
A: Yes! Create a project for yourself and manage personal tasks.

**Q: How many team members can I add?**
A: Unlimited! Add as many team members as needed.

**Q: What happens to tasks if I delete a project?**
A: All tasks are automatically deleted with the project.

**Q: Can I export tasks?**
A: Currently export via browser tools. Future feature planned.

**Q: Is my data secure?**
A: Yes! Passwords are hashed, JWT tokens used for auth, and CORS configured.
